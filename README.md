# Try C++, GLFW and Glad

> this repo is to learn how to build game engine and games from scratch using GLFW and OpenGL with C++

---

## To run this application on MacOs

### Install packages

```bash
brew install sdl2
brew install glfw
```

### Clean

```Bash
make clean
```

### Build

```Bash
make
```

### Run

```Bash
make run
```
